# Nur Theme for iTerm2

Nur is a color theme designed to provide a pleasant experience for long coding sessions. This iTerm2 version brings the warm, relaxing colors of Sardinian landscapes to your terminal.

## Installation

### Method 1: Direct Import
1. Download the theme files:
   - [nur-iterm2-dark.itermcolors](themes/nur-iterm2-dark.itermcolors)
   - [nur-iterm2-light.itermcolors](themes/nur-iterm2-light.itermcolors)
2. Open iTerm2
3. Go to **Preferences** (⌘+,)
4. Navigate to **Profiles** → **Colors**
5. Click on **Color Presets...** dropdown (bottom right)
6. Select **Import...**
7. Choose the downloaded `.itermcolors` file
8. Select the imported theme from **Color Presets...**

### Method 2: Command Line
```bash
# Clone the repository
git clone https://github.com/michelefenu/nur.git

# Open the theme files (they will auto-import to iTerm2)
open nur/iterm2/themes/nur-iterm2-dark.itermcolors
open nur/iterm2/themes/nur-iterm2-light.itermcolors
```

## Color Palette

### Nur Dark
- **Background**: #23272b (Ossidiana)
- **Foreground**: #e2dbc4 (Orbace)
- **Cursor**: #fc7261 (Corallo)
- **Selection**: #6e7f5c (Ulivo)

### Nur Light  
- **Background**: #f8f5ed (Orbace Chiaro)
- **Foreground**: #5a5e6b (Basalto)
- **Cursor**: #e54837 (Corallo Vivo)
- **Selection**: #5a6a4d (Ulivo Profondo)
