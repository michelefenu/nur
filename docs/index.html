Nur Theme

A Visual Studio Code and Cursor theme designed to provide a pleasant experience for long coding sessions in any programming language.

MIT License VS Marketplace Open VSX

Features

  • Warm, relaxing and accessible color palette - Designed for comfortable long coding sessions
  • Markup and Code highlighting optimized for web development - Enhanced syntax highlighting
  • Light and Dark modes - Choose the mode that suits your preference

Palette and Inspirations

Inspired by the vibrant terracotta sunsets, ancient stones, crystal-clear shores, and the rich heritage of the island of Sardinia, this theme offers a warm, relaxing, and accessible visual experience.
